# 1 单行注释

单行注释以＃字符开头。

```ruby
# This is single line comment.
```

# 2 多行注释

多行注释使用=begin开始并在行尾以=end为终点。

```ruby
=begin  
    This is  
    multi line comment  
    line 1 another line...
=end
```
