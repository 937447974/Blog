Certificate, Key, and Trust Services为管理证书、公钥私钥和信任策略提供了C语言接口。您可以在您的应用程序中使用这些服务：

1. 通过私钥确定身份；
2. 创建和请求证书；
3. 导入证书、密钥和身份；
4. 创建公钥-私钥对；
5. 代表信任策略。

**Concurrency Considerations**

在iOS上，该文章介绍的所有API都是线程安全的。
